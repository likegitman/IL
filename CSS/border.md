# Border

## Use
> border: 선크기 점선/실선 색상  
`border: 1px solid lightgrey;`  
`border: 1px dotted lightgrey;`

## Direction
```
border-top: 1px solid lightgrey;  
border-bottom: 1px solid lightgrey;  
border-right: 1px solid lightgrey;  
border-left: 1px solid lightgrey;  
```

## border-radius
> 선의 둥글기를 설정  
`border-radius: 50%/px;`
