# Conditional
* if 
```
const name = 'Mike';
if (name === 'Mike') {
  console.log('Oh! Mike!');
} else if (name === 'Jeny') {
  console.log('Hi Jeny');
} else {
  console.log('Who..?');
}
```

* switch-case
```
const name = 'Mike';
switch(name) {
  case 'Mike':
    console.log('Oh! Mike!');
    break;
  case 'Jeny':
    console.log('Hi Jeny');
    break;
  default:
    console.log('Who..?');
    break;
}
```
